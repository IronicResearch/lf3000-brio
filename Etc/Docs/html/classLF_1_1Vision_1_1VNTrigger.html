Detailed Description

Virtual base class for all trigger logic. VNTrigger objects are used in conjunction with VNHotSpots to determine if the hot spot was triggered. Triggering events are based on the type of trigger and the current agorithm the VNVisionMPI is using.

NOTE: For use with LeapTV applications ONLY.
