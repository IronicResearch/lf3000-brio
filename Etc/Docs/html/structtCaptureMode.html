Detailed Description

Three components: image format, image resolution, and video frame rate, determine the camera's capture mode. Note video frame rate is used by Video4Linux in units of time-per-frame, so the 'fps' fields are misnamed and actually referring to inverse fps when using numerator:denominator ratio.
