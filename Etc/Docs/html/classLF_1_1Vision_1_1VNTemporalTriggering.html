Detailed Description

NOTE: For use with LeapTV applications ONLY.

Virtual base class for all temporally related triggering logic. A VNTemporalTriggering operates in conjunction with a NVTrigger object within a VNCompoundTrigger to determine if a VNHotSpot object has been triggered both spatially and temporally.
