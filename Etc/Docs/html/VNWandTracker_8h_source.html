LF::Vision::VNWandTracker
The VNWandTracker class is a discrete algorithm used to track the end of the LF wand, a colored light source. The basic algorithm filters the input video frames based on a specific hue, based on the light color, then thresholds the resulting image based on saturation and intensity or brightness.
