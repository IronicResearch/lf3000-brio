Detailed Description

NOTE: For use with LeapTV applications only. VNHotSpotEventMessage is used to send information back to the application code when VNHotSpots are triggered. These messgaes get sent via an IEventListener implemented in the application code.

const VNHotSpot* LF::Vision::VNHotSpotEventMessage::GetHotSpot ( void  ) const

GetHotSpot returns the VNHotSpot associated with this message. If the event type for the listener is kVNHotSpotTriggeredEvent or kVNHotSpotTriggerChangeEvent this method will return a non-NULL value associated with the VNHotSpot. If the event type for the listener is kVNHotSpotGroupTriggeredEvent or kVNHotSpotGroupTriggerChangeEvent this method returns NULL.

Returns
A const pointer to the VNHotSpot associated with this message
