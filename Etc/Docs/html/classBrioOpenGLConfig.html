Detailed Description

Class for binding an OpenGL ES context with Brio DisplayMPI context. Class members export EGL context variables for calling EGL functions like eglSwapBuffers(), plus DisplayMPI display context handle for calling DisplayMPI functions like SetWindowPosition() and SetAlpha().

LF1000: The OpenGL ES rendering context is RGB565 format only. Separate 1D and 2D heaps are allocated from framebuffer memory for supporting vertex buffers and textures.

LF2000: The OpenGL ES rendering context is ARGB8888 format. Memory for vertex buffers and textures are allocated on demand.
